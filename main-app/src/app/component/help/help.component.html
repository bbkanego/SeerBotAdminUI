What are categories?

  • Categories allow you to define what kind of Bot you want to create.
  • You can use existing/pre-defined category to create your Bots.
  • If you want to create your own categories, then you can click on Create Category menu option and create the desired category. This link can be found under "Reference Data section.
  • Once a Category has been created, you can edit it as needed by clicking on the Search Category menu option.

All About Utterances & Intents

  • Intents define the intentions of the Human user interacting with your Bot.
  • Your Bot is able to understand Human interactions based on the Intents and Utterances that you provide.
  • You typically define an "Intent" like "GetOrder" or "CancelReservation and then define 15-20 utterances that satisfy the intent. This can be done using Add Intents menu option.
  • Utterance means what the Human user will ask the Bot when they have a specific intention. For example when the Human user wants to cancel a reservation, the user may say "I want to cancel my reservation" or "How can I cancel a booking. You provide these typical utterance's to the Bot so that the Bot can determine that the Human users intention is the intent "CancelReservation.
  • When you click on Add Intents you will need to enter the following information:
    1. The Intent Name
    2. Multiple utterance as required. We recommend at least 15-20 different variations of the way the Human users will ask question to the bot for the specific intent
    3. Provide response the bot should send back to the user when the bot understands the user's intent. You can include HTML in the response as well.
      Sure I can help you with that. You can reset or unlock your by clicking this link <a href='http://somewebsite.com/page'>Unlock Now<a>
    4. Provide response the bot should send back to the user when the bot partially understands the user's intent. You can include HTML in the response as well.
      Sorry, I do not understand. Are you looking for nut free food?|butt=Get Reservation&res=I need a reservation|butt=Specials Today &res=specials today|butt=Health Score&res=health score

Training Models

  • By now you must have got a good understanding of how to define Intents, Utterances and what they are.
  • Once you are done defining the Intents and Utterance for a specific category, you can now create trained models.
  • Trained Models are a dictionary that Bots use in order to understand human interactions and intentions.
  • You can create Trained Models by clicking on the link Create Trained Model"
  • Once a Trained Model has been created you edit by going to Search Trained Models"

What are those Bots?

  • Bots are automated customer service tools that allow you to answer customer's question/queries without human intervention.
  • You can set up the Bot to involve human intervention, like forwarding the customer to a live assistant based on certain conditions.
  • You can define Bots for each category. Then you can train your bot to understand Intents using different trained models. Refer the trained model section on what they are and how you can create them.

Create a Bot from scratch

Now that you have a good understanding of Utterances, Intents, Trained Models and Bots, lets look at an example where you can go about creating a new bot from scratch.
  • Step 1 - Add Category: Create a new category or use an existing predefined category. You can use the Create Category menu option to do so.
  • Step 2 - Add Intents & Utterances: Next add intents and utterance as required for the Category. You can use the menu option Add Intents to do so.
  • Step 3: - Create Model and Train: Next create a new model for your category and train the model using the intents/utterances defined in step 2. Use menu option Add Intents
  • Step 4: - Create Bot: Next create a new bot for your business using menu option Create Bot
  • Step 5: - Launch Bot in Test mode: Using menu option Launch Bot launch your bot. By default bot will launched in "Test" mode.
  • Step 6: - Test Your Bot: Next create a new bot for your business using menu option Test Bot
  • Step 7: - Launch Bot in Production mode: Once you are satisfied with the how your bot performs in test mode, you can promote the Bot to "Production" mode so that you can integrate the Bot on your website or app. Use menu option Test Bot to do so.
  • Step 8: - Monitor your Bot: Next create a new bot for your business using menu option Test Bot
